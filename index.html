Welcome to the Politeness Annotation Guide


Introduction

Politeness which can be defined as acting so as to take into account the feelings of others is key to human interactions, smoothing communication and social interaction.

Annotation Process Overview

  • Step 1: Evaluate the level of politeness: For each sentence, the annotator must determine the level of politeness of the sentence. For this he/she must answer: What do you consider to be the level of politeness of the sentence?
  • Step 2: Filter sentences: Researchers filter out statements that can be paraphrased as more polite or less polite.
  • Step 3: Paraphrase the sentence For each sentence, the annotator must re-write a less polite version of the sentence and a more polite version of the sentence.

    • Evaluate the level of politeness Task (AGREGAR UN SCREENSHOT ACA)

    For each sentence, the annotator must determine the level of politeness of the sentence. For this he/she must answer: What do you consider to be the level of politeness of the sentence? For example:

  • very impolite : Your performance in the match was bad, you need to train more.
  • somewhat impolite: Train more for the next game
  • neutral: Hey, you should train more for the next game.
  • somewhat polite: I think you should train more for the next game.
  • very polite : Sorry to bother you, but I think you should train more for the next game.

    • Paraphrase the sentence Task (AGREGAR UN SCREENSHOT ACA)

    Each sentence should be evaluated independently and should be paraphrased as naturally as possible, trying to modify only the level of politeness of the sentence and not the meaning of the sentence.

    General Rules

    Positive, negative, and multi-word rules that apply to all mention labels, including general spelling rules.

    Positive Rules

    Rules that specify strategies that can be used in paraphrasing:

    • When paraphrasing, natural language must be used, so the use of emojis, abbreviations or expressions used in informal contexts is permitted.
    • It is possible to add expressions in the first person to include the speaker's opinion even when the original sentence is in the second person. As seen in example 1 and 2 when adding ‚ÄúI think‚Äù or ‚ÄúI suggest‚Äù.
    • Expressions of gratitude can be included if the context allows it, for example ‚ÄúI appreciate‚Äù.
    • Expressions of apology can be included if the context requires it. As seen in example 1 when adding ‚ÄúSorry to bother you‚Äù.
    • It is possible to change the verbal mode of the sentence. As seen in example 3 when changing "Could you send me" to " Can you send me .."

    Negative Rules

    Rules that specify strategies that cannot be used when paraphrasing or actions that are not allowed:

    • The use of vulgar language or insults is not permitted. This type of vocabulary falls into the category of hate speech, which is outside the scope of this study.
    • The repeated use of the same strategy should be avoided. For example, adding ‚ÄúThanks‚Äù or ‚ÄúPlease‚Äù to the end of all sentences.
    • Overly wordy paraphrases that differ greatly from the length of the original example should be avoided.

    More polite paraphrase

    "You should read the document by tomorrow"

    • I think it would be great if you could read the document by tomorrow.

      Reasoning:
      • The original sentence is not very polite since it is an order given to another person. As a strategy we find that it is addressed in the 2nd person and using counterfactual mode
      • Paraphrasing is more polite since it is presented as a suggestion. The strategies found are beginning in the first person and the use of counterfactual mode.

    • I would appreciate it if you could read the document by tomorrow.

      Reasoning:
      • The paraphrase is more polite since it includes a gesture of deference at the beginning and the counterfactual mood is used.

    • Could you please read the document by tomorrow?

      Reasoning:
      • Paraphrasing is more polite since it is presented as a question in counterfactual mode. In addition, the strategy of adding "please" is used.

    • Sorry to bother you but I wonder if you could read the document by tomorrow?

      Reasoning:
      • Paraphrasing is more polite since it is presented as a question in counterfactual mode. Also it includes an apologizing at the beginning and asks about the possibility of performing the action instead of presenting it as an order.
